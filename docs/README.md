# AI 재무 플랫폼 - 디케이동신

## 개요
컬러강판(PCM) 수출 제조업체를 위한 AI 기반 재무 관리 플랫폼

## 대상 고객
- **회사명**: 디케이동신
- **업종**: 컬러강판 제조/수출
- **직원수**: 139명
- **특징**: 수출 비중 높음, ERP 사용 중 (더존 또는 SAP 추정)

## 핵심 가치
> "AI가 판단하고 제안하는 재무 플랫폼"

기존 ERP는 그대로 사용하고, ERP 데이터를 가져와서 AI 분석만 수행.
업무 프로세스 변경 없이 기존 시스템 위에 얹는 방식.

## 아키텍처

```
[ERP (더존/SAP)]
      ↓ 데이터 연동 (API 또는 엑셀)
[AI 재무 플랫폼]
      ├── 무역서류 OCR 파싱
      ├── 손익 분석 + AI 코멘트  
      ├── 채권/환율 관리
      ├── 시뮬레이션
      └── 보고서 자동 생성
```

## ERP 연동형 전략

### ERP에서 가져오는 것 (데이터 소스)
- 매출 데이터
- 매입/원가 데이터
- 채권/채무 데이터
- 계정과목 마스터
- 재무제표 (손익계산서, 재무상태표)

### 우리가 하는 것 (AI 플랫폼)
- 무역서류 OCR → ERP 연동용 데이터 생성
- ERP 데이터 기반 AI 분석/코멘트
- 시뮬레이션
- 보고서 자동 작성
- 이상 탐지 알림

### 안 하는 것 (ERP 영역)
- ❌ 회계전표 입력/관리
- ❌ 총계정원장/보조원장
- ❌ 재무제표 생성
- ❌ 계정과목 마스터 관리
- ❌ 결산 처리

## 기술 스택
- **Frontend**: React + TypeScript + Tailwind CSS
- **Backend**: FastAPI (Python)
- **AI**: Claude API (Anthropic)
- **OCR**: Claude Vision API
- **DB**: PostgreSQL (운영) / SQLite (데모)

## 문서 구조

- `PROMPT_TEMPLATES.md` - AI 프롬프트 템플릿
