## 기술 유형별 분류

### Vision + LLM (OCR)
서류 이미지에서 데이터 추출

| 대상 서류 | 추출 항목 |
|----------|----------|
| Commercial Invoice | 인보이스번호, 날짜, 거래처, 품목, 수량, 단가, 금액, 통화 |
| B/L (선하증권) | B/L번호, 선적일, 출발항, 목적항, 수량, 선박명 |
| Packing List | 품목, 박스수, 중량, 규격 |
| L/C (신용장) | 신용장번호, 금액, 유효기간, 조건 |
| 수입 인보이스 | 공급업체, 품목, 단가, 금액 |

### LLM (분석/생성)
텍스트 기반 분석 및 코멘트 생성

| 기능 | 입력 | 출력 |
|------|------|------|
| 손익 원인 분석 | 손익 데이터 JSON | 원인 분석 텍스트 |
| 경영진 코멘트 | 손익 데이터 JSON | 보고서용 코멘트 |
| 시뮬레이션 권장 | 시뮬레이션 결과 | 대응 방안 제안 |
| 서류 불일치 원인 | 불일치 항목 | 원인 추론 |
| 보고서 작성 | 재무 데이터 | 전체 보고서 |

### 룰 + LLM (탐지/알림)
규칙 기반 탐지 + LLM 설명

| 기능 | 룰 (탐지) | LLM (설명) |
|------|----------|-----------|
| 원가 변동 알림 | 전월 대비 5% 이상 변동 | "냉연강판 단가 8% 상승으로 인한 원가율 증가" |
| 연체 알림 | 결제예정일 초과 | "A바이어 60일 연체, 총 미수금 $50,000" |
| 환율 급변동 | 전일 대비 2% 이상 변동 | "원/달러 환율 2.5% 급등, 수출채권 환산이익 예상" |
| 매출 급감 | 전월 대비 20% 이상 감소 | "건재용 매출 25% 감소, 건설경기 둔화 영향 추정" |

### 매칭 알고리즘
AI 보조 자동 매칭

| 기능 | 매칭 기준 |
|------|----------|
| 인보이스-입금 대사 | 금액, 거래처명, 날짜 유사도 |
| 서류 대사 | 인보이스번호, 수량, 금액 |
